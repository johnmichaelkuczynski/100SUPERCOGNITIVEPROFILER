Testing Logic and Set Theory Mathematical Notation

Logical Connectives and Operators

Conjunction: \(P \land Q\) (interpreted as \(P\) AND \(Q\))

Disjunction: \(P \lor Q\) (interpreted as \(P\) OR \(Q\))

Negation: \(\neg P\) (interpreted as NOT \(P\))

Implication: \(P \to Q\) (interpreted as \(P\) implies \(Q\))

Set Theory Notation

Set membership: \(x \in A\) and \(x \notin A\)

Subset relations: \(A \subseteq B\) and \(A \subset B\)

Set operations: \(A \cup B\), \(A \cap B\), \(A \setminus B\)

Cartesian product: \(A \times B\)

Quantifiers

Universal quantifier: \(\forall x \in \mathbb{R}\)

Existential quantifier: \(\exists x \in \mathbb{N}\)

Unique existence: \(\exists! x\)

Real Number System

Natural numbers: \(\mathbb{N} = \{1, 2, 3, ...\}\)

Integers: \(\mathbb{Z} = \{..., -2, -1, 0, 1, 2, ...\}\)

Rational numbers: \(\mathbb{Q} = \{\frac{p}{q} : p, q \in \mathbb{Z}, q \neq 0\}\)

Real numbers: \(\mathbb{R}\)

Complex Expressions from Document

De Morgan's Laws: \(\neg(P \lor Q) \leftrightarrow (\neg P \land \neg Q)\)

Modus Ponens: \((P \to Q) \land P \vdash Q\)

Modus Tollens: \((P \to Q) \land \neg Q \vdash \neg P\)

Advanced Set Theory

Power set: \(\mathcal{P}(A) = \{B : B \subseteq A\}\)

Cardinality: \(|A|\)

Infinite sets: \(|\mathbb{N}| = \aleph_0\)

Functions and Relations

Function definition: \(f: A \to B\)

Injective: \(\forall x_1, x_2 \in A, f(x_1) = f(x_2) \Rightarrow x_1 = x_2\)

Surjective: \(\forall y \in B, \exists x \in A, f(x) = y\)

Logical Equivalences

Absorption Laws: \(P \lor (P \land Q) \equiv P\)

Distributive Laws: \(P \land (Q \lor R) \equiv (P \land Q) \lor (P \land R)\)

Complex Mathematical Statements

Completeness of Real Numbers: Every non-empty set of real numbers that is bounded above has a least upper bound (supremum).

Formally: \(\forall S \subseteq \mathbb{R}, S \neq \emptyset \land (\exists M \in \mathbb{R}, \forall x \in S, x \leq M) \Rightarrow \exists \sup S \in \mathbb{R}\)
